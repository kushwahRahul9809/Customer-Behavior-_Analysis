# Customer-Behavior-_Analysis


# Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases.
The objective is to discover customer spending patterns, product preferences, subscription
behavior, and business insights that support strategic decision-making.
# Dataset Summary
Rows: 3,900
Columns: 18
# Features: 
Customer demographics,
purchase details, 
shopping behavior, 
subscriptions, discounts,
shipping preferences,
and ratings.
# Data Cleaning & Preprocessing
• Missing value handling for Review Rating.
• Column standardization to snake_case.
• Feature engineering for age groups and purchase frequency.
• PostgreSQL integration for SQL analysis.
# Business Analysis Performed
1. Revenue by Gender
2. High-Spending Discount Users
3. Top 5 Products by Rating
4. Shipping Type Comparison
5. Subscribers vs Non-Subscribers
6. Discount-Dependent Products
7. Customer Segmentation
8. Top Products per Category
9. Repeat Buyers & Subscription Analysis
10. Revenue by Age Group
# Dashboard & Visualization
Power BI was used to create an interactive dashboard for customer demographics, revenue trends,
product performance, subscription insights, and shipping preferences.
Business Recommendations
• Promote subscriptions through exclusive benefits.
• Implement customer loyalty programs.
• Optimize discount policies.
• Highlight top-rated products.
• Use targeted marketing for high-value customer groups
